Description

JpcapDumper is a graphical Java application to display packets captured by Jpcap.

 

Features
  • Real time capturing
  • Can read and display dump file created by tcpdump
  • Supports Ethernet, IPv4/v6, TCP, UDP, ICMP, HTTP

 

What's New
